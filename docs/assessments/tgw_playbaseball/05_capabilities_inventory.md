# Ecommerce Capabilities Inventory

## Feature Comparison Matrix

| Feature Category  | Feature                     | tgw.com (Legacy)   | playbaseball.com (Modern) | Notes                                                                                                                                       |
| :---------------- | :-------------------------- | :----------------- | :------------------------ | :------------------------------------------------------------------------------------------------------------------------------------------ |
| **Search**        | **Autosuggest / Typeahead** | ❌ No               | ✅ Yes                     | **playbaseball.com** offers instant product suggestions while typing.                                                                       |
|                   | **Search Results Page**     | ✅ Yes              | ✅ Yes                     | Both sites support grid views.                                                                                                              |
| **Catalog (PLP)** | **Faceted Filtering**       | ✅ Yes              | ✅ Yes                     | Both support standard filters (Brand, Price, etc.).                                                                                         |
|                   | **Sorting**                 | ✅ Yes              | ✅ Yes                     | Standard sorting options available on both.                                                                                                 |
| **Product (PDP)** | **Reviews / Ratings**       | ✅ Yes              | ✅ Yes                     | Both display star ratings and review counts.                                                                                                |
|                   | **Image Zoom**              | ✅ Yes              | ✅ Yes                     | Both support click-to-zoom.                                                                                                                 |
|                   | **Stock Status**            | ✅ Yes ("In Stock") | ✅ Yes (Implied)           | **tgw.com** is more explicit with "In Stock" messaging.                                                                                     |
|                   | **Wishlist**                | ✅ Yes              | ✅ Yes                     | Both support saving items for later.                                                                                                        |
| **Multi-site**    | **Shared Cart**             | ❌ No               | ✅ Yes                     | **playbaseball.com** shares cart with `playsoftball.playbaseball.com`. "ALL BALL. ONE CART." messaging.                                     |
| **Account**       | **Sign In / Register**      | ✅ Yes              | ✅ Yes                     | Standard account management.                                                                                                                |
|                   | **Order Tracking**          | ✅ Yes (Footer)     | ⚠️ Hidden                  | **tgw.com** has a clear "Order Status" link in the footer. **playbaseball.com** likely buries this in "Customer Service" or requires login. |
| **Marketing**     | **Email Signup**            | ✅ Yes              | ✅ Yes                     | Both have footer signup forms.                                                                                                              |

## Key Observations
1.  **playbaseball.com** has autosuggest search
2.  **tgw.com** provides clearer stock status messaging
3.  **tgw.com** has more accessible order tracking (footer link)
4.  **playbaseball.com** has shared cart across multi-site architecture
